
# Contributor Code of Conduct

We expect all contributors to behave respectfully. Be welcoming, patient and constructive.

Unacceptable behaviour includes harassment, discrimination, abusive language, or personal attacks. If you experience or witness unacceptable behaviour, please report it to the maintainers via a private GitHub issue or contact the repository owner.

For ownership information see: [.github/CODEOWNERS](.github/CODEOWNERS)
